# FUTURE_DS_01
Task 1 - Sales Data Analysis using Python (Pandas, Matplotlib). 
This repository contains Task 1 of my Data Science & Analytics Internship. In this project, I performed sales data analysis using Python, Pandas, and Matplotlib. The analysis includes data cleaning, monthly sales trend visualization, category-wise and sub-category-wise performance analysis, and extraction of meaningful business insights. This project demonstrates my ability to work with real-world datasets and generate actionable insights.
